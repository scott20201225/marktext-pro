# Environment

| Name                         | Description                                                 |
| ---------------------------- | ----------------------------------------------------------- |
| `MARKTEXTPRO_DEBUG`             | Enable debug mode.                                          |
| `MARKTEXTPRO_DEBUG_KEYBOARD`    | Print more keyboard information when debug mode is enabled. |
| `MARKTEXTPRO_ERROR_INTERACTION` | Never show the error dialog to report bugs.                 |
| `MARKTEXTPRO_PANDOC`            | Overwrite the pandoc path.                                  |

## Development

| Name                     | Description                                                  |
| ------------------------ | ------------------------------------------------------------ |
| `MARKTEXTPRO_EXIT_ON_ERROR` | Exit on the first error or exception that occurs.            |
| `MARKTEXTPRO_IS_STABLE`     | **Please don't use this!** Used to identify stable releases. |
