#### Image Uploader Configuration

##### PicGo

PicGo is a CLI tool that uploads images to various cloud providers. See the [PicGo documentation](https://picgo.github.io/PicGo-Doc/en/guide/) for installation, configuration, and supported backends.

Once PicGo is installed and configured, point MarkTextPro at the PicGo executable in *Preferences → Image* and choose `upload` as the *image insert action* (or use it on demand from the image overlay). PicGo is the only built-in uploader; the legacy GitHub uploader was removed in 0.19.
