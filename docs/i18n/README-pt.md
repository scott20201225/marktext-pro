## Recursos

- Visualização em tempo real (WYSIWYG) e interface limpa e simples para uma experiência de escrita sem distrações.
- Suporte a [CommonMark Spec](https://spec.commonmark.org), [GitHub Flavored Markdown Spec](https://github.github.com/gfm/) e suporte seletivo a [Pandoc markdown](https://pandoc.org/MANUAL.html#pandocs-markdown).
- Extensões Markdown como expressões matemáticas (KaTeX), front matter e emojis.
- Suporte a atalhos de parágrafo e estilo inline para melhorar sua eficiência de escrita.
- Exportação de arquivos **HTML** e **PDF**.
- Vários [temas](../end-user/THEMES.md): **Cadmium Light**, **Material Dark** etc.
- Vários modos de edição: **Modo Código-Fonte**, **Modo Máquina de Escrever**, **Modo Foco**.
- Cole imagens diretamente da área de transferência.

#### macOS

Você pode baixar o último `marktextpro-%version%.dmg` da [página de releases](https://github.com/scott20201225/marktext-pro/releases/latest) ou instalar o MarkTextPro usando o [**homebrew cask**](https://github.com/caskroom/homebrew-cask). Para usar o Homebrew-Cask você só precisa ter o [Homebrew](https://brew.sh/) instalado.

```bash
brew install --cask marktextpro
```

#### Windows

Simplesmente baixe e instale o MarkTextPro via assistente de configuração (`marktextpro-setup-%version%.exe`) e escolha instalar por usuário ou para toda a máquina. Alternativamente, instale o MarkTextPro usando um gerenciador de pacotes como [Chocolatey](https://chocolatey.org/) ou [Winget](https://docs.microsoft.com/en-us/windows/package-manager/winget/).

Para usar o Chocolatey, você precisa ter o [Chocolatey](https://chocolatey.org/install) instalado:

```bash
choco install marktextpro
```

Para usar o Winget, você precisa ter o [Winget](https://docs.microsoft.com/en-us/windows/package-manager/winget/#install-winget) instalado:

```bash
winget install marktextpro
```

#### Linux

Por favor siga as [instruções de instalação do Linux](../end-user/LINUX.md).

#### Outros

Todos os binários para Linux, macOS e Windows podem ser baixados da [página de releases](https://github.com/scott20201225/marktext-pro/releases/latest). Se uma versão não estiver disponível para o seu sistema, por favor abra um [issue](https://github.com/scott20201225/marktext-pro/issues).
