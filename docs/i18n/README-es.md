## Características

- Vista previa en tiempo real (WYSIWYG) e interfaz limpia y simple para una experiencia de escritura sin distracciones.
- Compatible con [CommonMark Spec](https://spec.commonmark.org), [GitHub Flavored Markdown Spec](https://github.github.com/gfm/) y soporte selectivo de [Pandoc markdown](https://pandoc.org/MANUAL.html#pandocs-markdown).
- Extensiones Markdown como expresiones matemáticas (KaTeX), front matter y emojis.
- Soporte para atajos de párrafo y estilo en línea para mejorar tu eficiencia de escritura.
- Exportación de archivos **HTML** y **PDF**.
- Varios [temas](../end-user/THEMES.md): **Cadmium Light**, **Material Dark**, etc.
- Varios modos de edición: **Modo Código Fuente**, **Modo Máquina de Escribir**, **Modo Enfoque**.
- Pegado de imágenes directamente desde el portapapeles.

#### macOS

Puedes descargar el último `marktextpro-%version%.dmg` desde la [página de releases](https://github.com/scott20201225/marktext-pro/releases/latest) o instalar MarkTextPro usando [**homebrew cask**](https://github.com/caskroom/homebrew-cask). Para usar Homebrew-Cask solo necesitas tener [Homebrew](https://brew.sh/) instalado.

```bash
brew install --cask marktextpro
```

#### Windows

Simplemente descarga e instala MarkTextPro mediante el asistente de configuración (`marktextpro-setup-%version%.exe`) y elige si instalarlo por usuario o para todo el sistema. Alternativamente, instala MarkTextPro usando un gestor de paquetes como [Chocolatey](https://chocolatey.org/) o [Winget](https://docs.microsoft.com/en-us/windows/package-manager/winget/).

Para usar Chocolatey, necesitas tener [Chocolatey](https://chocolatey.org/install) instalado:

```bash
choco install marktextpro
```

Para usar Winget, necesitas tener [Winget](https://docs.microsoft.com/en-us/windows/package-manager/winget/#install-winget) instalado:

```bash
winget install marktextpro
```

#### Linux

Por favor sigue las [instrucciones de instalación de Linux](../end-user/LINUX.md).

#### Otros

Todos los binarios para Linux, macOS y Windows se pueden descargar desde la [página de releases](https://github.com/scott20201225/marktext-pro/releases/latest). Si no hay una versión disponible para tu sistema, por favor abre un [issue](https://github.com/scott20201225/marktext-pro/issues).
