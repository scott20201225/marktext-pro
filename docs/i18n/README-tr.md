## Özellikler

- Gerçek zamanlı önizleme (WYSIWYG) ve temiz, basit arayüz ile odaklanmış yazma deneyimi.
- [CommonMark Spec](https://spec.commonmark.org), [GitHub Flavored Markdown Spec](https://github.github.com/gfm/) desteği ve seçici [Pandoc markdown](https://pandoc.org/MANUAL.html#pandocs-markdown) desteği.
- Matematiksel ifadeler (KaTeX), front matter ve emoji gibi Markdown uzantıları.
- Yazma verimliliğinizi artırmak için paragraf ve satır içi stil kısayolları.
- **HTML** ve **PDF** dosyası çıktısı.
- Çeşitli [temalar](../end-user/THEMES.md): **Cadmium Light**, **Material Dark** vb.
- Çeşitli düzenleme modları: **Kaynak Kodu modu**, **Daktilo modu**, **Odak modu**.
- Görüntüleri doğrudan panodan yapıştırın.

#### macOS

[Sürümler sayfasından](https://github.com/scott20201225/marktext-pro/releases/latest) en son `marktextpro-%version%.dmg`'yi indirebilir veya [**homebrew cask**](https://github.com/caskroom/homebrew-cask) kullanarak MarkTextPro'i kurabilirsiniz. Homebrew-Cask kullanmak için [Homebrew](https://brew.sh/)'ün yüklü olması gerekir.

```bash
brew install --cask marktextpro
```

#### Windows

MarkTextPro'i kurulum sihirbazı (`marktextpro-setup-%version%.exe`) aracılığıyla indirip kurun ve kullanıcı başına mı yoksa sistem genelinde mi kurulacağını seçin. Alternatif olarak [Chocolatey](https://chocolatey.org/) veya [Winget](https://docs.microsoft.com/en-us/windows/package-manager/winget/) gibi bir paket yöneticisi kullanabilirsiniz.

Chocolatey kullanmak için [Chocolatey](https://chocolatey.org/install)'nin yüklü olması gerekir:

```bash
choco install marktextpro
```

Winget kullanmak için [Winget](https://docs.microsoft.com/en-us/windows/package-manager/winget/#install-winget)'in yüklü olması gerekir:

```bash
winget install marktextpro
```

#### Linux

Lütfen [Linux kurulum talimatlarını](../end-user/LINUX.md) takip edin.

#### Diğer

Linux, macOS ve Windows için tüm ikili dosyalar [sürümler sayfasından](https://github.com/scott20201225/marktext-pro/releases/latest) indirilebilir. Sisteminiz için bir sürüm mevcut değilse lütfen bir [issue](https://github.com/scott20201225/marktext-pro/issues) açın.
